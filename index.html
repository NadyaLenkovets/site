Технологии

Как я работаю.

Вёрстка кроссбраузерная, валидная, адаптивная, т.е. мои сайты красиво и правильно отображаются во всех популярных браузерах (Opera, Safari, Chrome и др.) и на любых устройствах: компьютеры, планшеты, смартфоны.

Код пишу понятно и продуманно, использую методологию БЭМ, семантические теги.

Верстаю из PSD, Figma, Zeplin, Avocode, Adobe XD.

Я не работаю на конструкторах сайтов.

Проверяю свою работу на валидаторе.

Что я использую:

  • HTML (HTML5)
  • CSS (CSS3)
  • Flexbox/Grid CSS
  • jQuery
  • Javascript
  • Препроцессоры LESS/SASS
  • SVG спрайты
  • Иконочные шрифты
  • Нестандартные шрифты (google fonts, @font-face)
  • Анимация сайтов
